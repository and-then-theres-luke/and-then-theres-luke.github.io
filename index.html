Projects

KitchenQuest

KitchenQuest is the cap-stone project I completed as part of Coding Dojo's Python Boot-Camp. It's a cooking app that's not an app, and it's not really about cooking either. KitchenQuest is my attempt to gamify inventory management in the kitchen. Use spells (ingredients) to defeat bosses (recipes) and gain experience (that's still experience). KitchenQuest was designed with a vanilla JavaScript frontend, Python with Flask on the back-end, and MySQL for database interactions. KitchenQuest is still in development, and the framework rests heavily on interactions with the Spoonacular API.
